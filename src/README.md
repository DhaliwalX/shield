# SHIELD

A complete autonomous solution to control swarm bots.

### A hell lot of dependencies
SHIELD depends on the following projects for building,

 - OpenCV
 - pthreads
 - libfreetype
 - OpenGL
 - GLUT
 - SDL2
 - zlib
 - libpng
 - libjpeg
 - Boost (specifically asio)
 - Google Skia library
 - libfontconfig
